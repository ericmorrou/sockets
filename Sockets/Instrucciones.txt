Demo Instructions para el Jueves
Here is a step-by-step guide to demoing this to your professor:

Abre dos terminales (consolas de comandos) y navega a la carpeta de tu escritorio:

cmd
cd Desktop\Sockets
Compila el código (si no lo has hecho ya):

cmd
javac ChatServer.java ChatClient.java
En la primera terminal, arranca el servidor:

cmd
java ChatServer
Verás el mensaje: Arrancando Servidor de Chat básico en el puerto 12345...

En la segunda terminal, arranca el cliente:

cmd
java ChatClient
Verás: ¡Conectado exitosamente al servidor! y te dará la bienvenida.

Interactúa con ambas ventanas escribiendo mensajes en la consola y dándole a Enter. Puedes ver cómo cada usuario recibe los mensajes sin bloquearse.

Muestra el cierre ordenado. Escribe EXIT desde cualquiera de las dos ventanas y pulsa enter. Verás varios mensajes por consola indicando que se ha cerrado todo exitosamente (Socket de cliente cerrado, etc) y el programa finalizará automáticamente.

¡Listo! Con esto tendrás un 10.
